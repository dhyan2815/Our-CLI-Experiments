# Assets Folder

Place your assets here for use in carousel slides.

## Folder Structure

```
assets/
├── headshot.png      ← REQUIRED for CTA slide (400x400px, square)
├── logos/            ← Brand logos (PNG with transparency)
└── icons/           ← Icon PNGs for visual elements
```

## Requirements

### headshot.png
- Used on final CTA slide
- Size: 400×400px recommended
- Format: PNG with transparency or solid background
- Name: must be `headshot.png`

### logos/
- Drop brand logo files here
- PNG preferred, transparency supported

### icons/
- Any custom icons for slide visuals
- No emoji — use PNG or reference in prompts

---

**Note:** If assets are missing, the carousel skill will use placeholders or skip those elements.
