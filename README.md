# DP's CLI Experiments

Welcome to my experimental playground for CLI-based tools, AI agents, and workflow automations. This repository is dedicated to exploring the capabilities and scaling possibilities of various command-line interfaces and their associated "skills" or "mcp" integrations.

## 🚀 Core Focus
This directory serves as a central hub for my experiments with:
- **Claude Code:** Testing agentic coding capabilities and CLI interactions.
- **Open Code:** Exploring open-source CLI-driven development.
- **Gemini CLI:** Scaling tasks and leveraging Google's Gemini models directly from the terminal.
- **n8n Workflows:** Designing and testing complex automation logic (exported as JSON).
- **Excalidraw Diagrams:** Visualizing architecture and logic flows for client presentations.

## 📂 Project Structure
- `workflows/`: Exported n8n workflow JSONs.
- `excalidraw diagrams/`: Visual documentation of system architectures.
- `project markdowns/`: Documentation and detailed guides for specific experiments.
- `.claude/` & `.opencode/`: Configuration and settings for CLI tools.

## 🛠 Experimental Nature
Everything in this repository is highly experimental. I use this space to push the boundaries of what CLI tools can do in a modern software engineering environment.
